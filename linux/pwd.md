# pwd

**pwd = print working directory**  
查看默认工作目录的完整路径
通常在不太确定当前位置时，使用pwd来判定当前目录在文件系统内的确切位置。

## 语法
```
pwd [选项]
```

## 参数
参数 | 说明
--|--
-L, --logical| 有链接文件时，直接显示链接文件的路径，(不加参数时默认此方式) 
-p, --physical| 有链接文件时，不使用链接路径，不管有几层，直接显示链接文件最终指向的文件
--help|显示帮助信息
--version|显示版本信息


## 示例
```
# pwd 查看当前目录
[root@localhost ~]# pwd
/root
```
